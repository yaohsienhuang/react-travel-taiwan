# Getting Started with travel_kaohsiung_react
 
# 分頁
- 景點列表 : 可依照區域選擇景點，default:全部
- 瀏覽紀錄 : 紀錄瀏覽過的景點
- 我的最愛 : 景點細節資料中，按下愛心即可將景點加入到`我的最愛`
- 我的成就 : 景點細節資料中，按下愛心即可將景點加入到`已去過的名單`
